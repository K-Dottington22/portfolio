# Kalyb Williams Cybersecurity Portfolio

This is a GitHub Pages portfolio for Kalyb Williams.

## How to upload

1. Open the ZIP file.
2. Upload the contents of the folder to the root of your `portfolio` repository.
3. Do not upload the ZIP file itself.
4. Do not upload the folder as a folder. Upload the files inside it.
5. In GitHub, go to Settings, then Pages.
6. Set Source to Deploy from a branch.
7. Set Branch to main and Folder to root.
8. Wait two to five minutes.

Your site should be live at:

https://k-dottington22.github.io/portfolio/

## Important

Your repository root should show `index.html` directly. If GitHub shows `PNG`, `IHDR`, or random symbols when visiting your site, the wrong file was uploaded as `index.html`.
